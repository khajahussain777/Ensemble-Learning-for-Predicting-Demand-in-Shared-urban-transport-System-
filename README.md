# Ensemble Learning for Predicting Demand in Shared Urban Transport

## Overview
This project predicts demand in shared urban transportation systems using machine learning and ensemble learning techniques. The objective is to forecast demand patterns and support efficient resource allocation.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib
- Joblib

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Demand prediction using ensemble models
- Model persistence with Joblib
- Interactive Streamlit dashboard

## Machine Learning Models
- Random Forest Regressor
- Gradient Boosting Regressor
- Ensemble Learning Framework

## Project Structure

```text
models/
│── train_model.py

app.py
requirements.txt
README.md
```

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
